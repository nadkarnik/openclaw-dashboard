# Contributing to OpenClaw Dashboard

Thank you for your interest in contributing! 🎉

## Getting Started

1. **Fork the repository** on GitHub
2. **Clone your fork** locally
3. **Install dependencies**: `npm install`
4. **Start the dev server**: `npm start`
5. **Make your changes** and test them
6. **Commit with clear messages**
7. **Push to your fork**
8. **Open a Pull Request**

## Development Setup

```bash
git clone https://github.com/YOUR-USERNAME/openclaw-dashboard.git
cd openclaw-dashboard
npm install
npm start
```

Visit http://localhost:3456 to see your changes.

## Code Style

- Use **2 spaces** for indentation
- Follow existing naming conventions
- Comment complex logic
- Keep functions small and focused

## What to Contribute

We welcome:
- 🐛 **Bug fixes**
- ✨ **New features** (please open an issue first to discuss)
- 📝 **Documentation improvements**
- 🎨 **UI/UX enhancements**
- 🌐 **Translations**
- ⚡ **Performance improvements**

## Testing

Before submitting:
1. Test with OpenClaw running
2. Check both light and dark themes
3. Verify mobile responsiveness
4. Ensure no console errors

## Commit Messages

Use clear, descriptive commit messages:

```
feat: Add export history to JSON
fix: Correct token calculation display
docs: Update installation instructions
style: Improve dark mode contrast
```

## Questions?

Open an issue or reach out on the [OpenClaw Discord](https://discord.com/invite/clawd)!
